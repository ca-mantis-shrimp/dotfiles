# Personal Config Management

you can mostly just read the [README](README.md) for the general cover of the config but what you will need to know is that this is primarily managed through chezmoi so one the one hand you have a layer between you and the machine, on the other hand, the internal complexity is higher due to the way that the framework around chezmoi works and requires a unixy structure of files, folders, and templates to make things work

more importantly, use this to refine your own toolbox as well as mine, this is where we can build not only customizations that transcend a single computer, but can add the layers we need to understand the subjects properly and get stuff structured like it needs to be structured
